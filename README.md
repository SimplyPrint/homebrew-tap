# SimplyPrint Homebrew Tap

Homebrew formulae and casks for SimplyPrint software.

## Installation

```bash
brew tap simplyprint/tap
```

## Available Software

### NFC Agent

NFC card reader agent for SimplyPrint 3D printer management.

```bash
brew install --cask nfc-agent
```

Or install in one command:

```bash
brew install --cask simplyprint/tap/nfc-agent
```

## Updating

```bash
brew update
brew upgrade --cask nfc-agent
```

## Uninstalling

```bash
brew uninstall --cask nfc-agent
```
